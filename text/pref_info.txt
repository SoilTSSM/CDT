This application needs Tcl/Tk to work properly.
You can download it from ActiveState's websites 
http://www.activestate.com/activetcl/downloads 

On Linux and OS X:
If you installed Tcl/Tk from the system's software manager. You have to install separately this two libraries: Tktable and BWidgets.
You can download Tktable here:
http://sourceforge.net/projects/tktable/files/tktable/2.10/
and BWidgets:
http://sourceforge.net/projects/tcllib/files/BWidget/1.9.5/
If you downloaded Tcl/Tk from ActiveState's websites, you only need to specify the Tktable  and BWidget installation path.

On Windows:
Install the exe file, and specify the installation directory of Tcl/Tk.
