Download data

Download satellite rainfall estimates data (RFE)

The menu Data Preparation ‣ Download RFE data   allows to download a satellite-based rainfall products from Tropical Applications of Meteorology using SATellite and Climate Hazards Group InfraRed Precipitation.


  1. Select the RFE product to download.
    • 10-DAYS TAMSATv3: TAMSAT version 3 dekadal rainfall
    • 10-DAYS TAMSATv2: TAMSAT version 2 dekadal rainfall
    • 10-DAYS CHIRPSv2.0: CHIRP with Station version 2.0 dekadal rainfall
    • 10-DAYS CHIRPv1.0: CHIRP version 1.0 dekadal rainfall
    • DAILY TAMSATv3: TAMSAT version 3 daily rainfall
    • DAILY TAMSATv2: TAMSAT version 2 daily rainfall
    • DAILY CHIRPSv2.0: CHIRP with Station version 2.0 daily rainfall
  2. Enter the start and end date of the data you want to download.
  3. Enter the bounding box of the region of interest.
  4. Enter the full path to the directory to save the downloaded data, or browse it from the right button   …    .

When everything is specified, click   Download     to start the download.

OUTPUT: Under the directory you specified to save the result (4), directories have been created according to the selected data source.

  • 10-DAYS TAMSAT version 2 and 3, 2 directories: Dekad_TAMSAT_Africa containing the dekadal rainfall data over Africa and Dekad_TAMSAT_Extracted containing the dekadal rainfall data extracted over the specified area.
  • DAILY TAMSAT version 2 and 3, 2 directories: Daily_TAMSAT_Africa containing the daily rainfall data over Africa and Daily_TAMSAT_Extracted containing the daily rainfall data extracted over the specified area.
  • 10-DAYS CHIRPSv2.0, one directory Dekad_CHIRPS_Extracted containing the dekadal rainfall data extracted over the specified area.
  • 10-DAYS CHIRPv1.0, one directory Dekad_CHIRP_Extracted containing the dekadal rainfall data extracted over the specified area.
  • DAILY CHIRPSv2.0, one directory Daily_CHIRPS_Extracted containing the daily rainfall data extracted over the specified area.

Download Digital Elevation Model data (DEM)

The menu Data Preparation ‣ Download DEM   allows to download a Digital Elevation Model.


  1. Enter the bounding box of your area of interest.
  2. Enter the full path to the directory to save the downloaded DEM, or browse it from the right button   …    .

When everything is specified, click   Download     to start the download.

OUTPUT: The downloaded data are stored in the directory you specified to save the result (2), there are 3 files:

  • DEM_1_Arc-Minute.nc: 1 Arc-Minute relief model data extracted over your area of interest. Data source ETOPO1
  • DEM_2_Arc-Minute.nc: 2 Arc-Minute gridded elevation and bathymetric data extracted over your area of interest. Data source ETOPO2v2
  • DEM_for_Merging.nc: elevation data in the form of a Digital Elevation Model regridded to 0.0375 x 0.0375 degree data spacing.

Download country boundary

The menu Data Preparation ‣ Download Country boundary   allows to download adminstrative boundaries for African countries from GADM database of Global Administrative Areas.


  1. Select the name of the country to download.
  2. Select the administrative subdivisions (level 0: the country boundary, 1: provinces, region or districts, …).
  3. Enter the full path to the directory to save the downloaded files, or browse it from the right button   …    .

When everything is specified, click   Download     to start the download.

OUTPUT: The downloaded shapefiles are stored in the directory you specified to save the result (3).
